# Crypto Bounty Hunter Workspace

This is our dedicated workspace to develop solutions, fix bugs, and build features for developer bounties to earn cryptocurrency.

## Workflow
1. Select an active bounty (Algora or Superteam Earn).
2. I will write the code/solution here.
3. You will submit the solution using your GitHub/Superteam account.
4. Once merged, you receive the cryptocurrency payout directly to your wallet!
